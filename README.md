## 🚦 Getting Started (Local Development)

To run NexusMines locally, follow these steps:

1. **Clone this repository**  
   `git clone https://github.com/itzzjustmateo/NexusMines`
2. **Install dependencies**  
   `bun install`
3. **Start the development server**  
   `bun run dev`
4. **Open your browser** at [localhost:3000](http://localhost:3000) (or the URL shown in your terminal).
